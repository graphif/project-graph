用于渲染节点右上角的按钮，该按钮用于展开或关闭节点的注释详情。

## API 方法

### `render(entity: Entity): void`

渲染指定 `entity` 的详情按钮。

- 如果 `entity` 没有详情内容，则不进行渲染。
- 检测鼠标是否悬停在详情按钮上。
- 如果鼠标悬停且 `entity` 未处于编辑详情状态，则在按钮上方显示提示文本 “点击展开或关闭节点注释详情”。
- 根据 `entity` 是否处于编辑详情状态，渲染不同的图标：
  - 如果正在编辑详情，显示 “✏️” 图标。
  - 否则，显示 “📃” 图标。
- 图标的字体大小会根据鼠标悬停状态和时间动态调整。
