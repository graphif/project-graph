## 功能亮点

- 基于 Rust 和 egui 构建，性能优越，支持大规模的节点图绘制
- 快速简单的操作方式，适合快速构思和迭代
- 跨平台支持，适用于 Windows、macOS、Linux、Android、Web，以及其他受 eframe 支持的平台
- 特有的文件格式 `.prg` (`application/vnd.project-graph`)，便于项目文件的管理和分享
